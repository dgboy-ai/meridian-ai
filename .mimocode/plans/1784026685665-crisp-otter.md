## Priority 1: Core Graph Algorithms (High Impact)

### 1.1 Multi-Hop BFS Lineage Traversal
**File:** `backend/stats.py`
**Problem:** `traverse_lineage()` only processes depth=1. Real DataHub graphs have 5-10+ hops.
**Solution:** Implement proper BFS with `collections.deque` for arbitrary depth traversal.

```python
from collections import deque

def bfs_lineage(source_urn, lineage_graph, max_depth=10):
    """BFS traversal of lineage graph to arbitrary depth."""
    queue = deque([(source_urn, 0)])
    visited = set()
    result = {"nodes": [], "edges": [], "depths": {}}
    
    while queue:
        current, depth = queue.popleft()
        if current in visited or depth > max_depth:
            continue
        visited.add(current)
        result["nodes"].append(current)
        result["depths"][current] = depth
        
        for neighbor in lineage_graph.get(current, []):
            if neighbor not in visited:
                queue.append((neighbor, depth + 1))
                result["edges"].append((current, neighbor))
    
    return result
```

### 1.2 Topological Sort for Pipeline Ordering
**File:** `backend/stats.py` (new function)
**Problem:** Pipeline execution order not respecting dependencies.
**Solution:** Kahn's algorithm for topological sort.

```python
def topological_sort(lineage_graph):
    """Topological sort using Kahn's algorithm."""
    in_degree = {node: 0 for node in lineage_graph}
    for node, neighbors in lineage_graph.items():
        for neighbor in neighbors:
            in_degree[neighbor] = in_degree.get(neighbor, 0) + 1
    
    queue = deque([node for node, deg in in_degree.items() if deg == 0])
    order = []
    
    while queue:
        node = queue.popleft()
        order.append(node)
        for neighbor in lineage_graph.get(node, []):
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    return order
```

### 1.3 Cycle Detection
**File:** `backend/stats.py` (new function)
**Problem:** No cycle detection in lineage graph.
**Solution:** DFS-based cycle detection.

```python
def has_cycle(lineage_graph):
    """Detect cycles in lineage graph using DFS."""
    WHITE, GRAY, BLACK = 0, 1, 2
    color = {node: WHITE for node in lineage_graph}
    
    def dfs(node):
        color[node] = GRAY
        for neighbor in lineage_graph.get(node, []):
            if color[neighbor] == GRAY:
                return True
            if color[neighbor] == WHITE and dfs(neighbor):
                return True
        color[node] = BLACK
        return False
    
    return any(dfs(node) for node, c in color.items() if c == WHITE)
```

## Priority 2: Search & String Algorithms

### 2.1 Trie for Entity Search
**File:** `backend/trie.py` (new file)
**Problem:** Entity search is O(n) linear scan.
**Solution:** Trie for O(m) prefix search (m = query length).

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False
        self.entity_urn = None

class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word, entity_urn):
        node = self.root
        for char in word.lower():
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end = True
        node.entity_urn = entity_urn
    
    def search_prefix(self, prefix):
        node = self.root
        for char in prefix.lower():
            if char not in node.children:
                return []
            node = node.children[char]
        return self._collect_all(node)
    
    def _collect_all(self, node):
        results = []
        if node.is_end:
            results.append(node.entity_urn)
        for child in node.children.values():
            results.extend(self._collect_all(child))
        return results
```

### 2.2 Binary Search for KS-Test
**File:** `backend/stats.py`
**Problem:** ECDF computation uses O(n) linear scan.
**Solution:** Binary search for O(log n) lookup.

```python
import bisect

def ks_test_binary(ref, cur):
    """KS-test with binary search for ECDF."""
    ref_sorted = sorted(ref)
    cur_sorted = sorted(cur)
    all_values = sorted(set(ref_sorted + cur_sorted))
    
    max_diff = 0
    n_ref, n_cur = len(ref_sorted), len(cur_sorted)
    
    for val in all_values:
        # Binary search for CDF values
        ref_cdf = bisect.bisect_right(ref_sorted, val) / n_ref
        cur_cdf = bisect.bisect_right(cur_sorted, val) / n_cur
        max_diff = max(max_diff, abs(ref_cdf - cur_cdf))
    
    return max_diff
```

## Priority 3: Performance Optimizations

### 3.1 Min-Heap for Top-K Results
**File:** `backend/semantic_search.py`
**Problem:** Full sort then slice for top-k.
**Solution:** `heapq.nlargest()` for O(n log k) instead of O(n log n).

```python
import heapq

def top_k_results(results, k=10):
    """Get top-k results using min-heap."""
    return heapq.nlargest(k, results, key=lambda x: x.similarity_score)
```

### 3.2 Union-Find for Lineage Components
**File:** `backend/union_find.py` (new file)
**Problem:** Repeated traversal to check connectivity.
**Solution:** Union-Find for O(α(n)) connectivity checks.

```python
class UnionFind:
    def __init__(self):
        self.parent = {}
        self.rank = {}
    
    def find(self, x):
        if x not in self.parent:
            self.parent[x] = x
            self.rank[x] = 0
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
    
    def connected(self, x, y):
        return self.find(x) == self.find(y)
    
    def get_component(self, x):
        root = self.find(x)
        return [node for node, parent in self.parent.items() if self.find(parent) == root]
```

### 3.3 Memoization for Root Cause Scoring
**File:** `backend/workers/root_cause.py`
**Problem:** O(n^2) scoring without caching.
**Solution:** `@functools.lru_cache` for memoization.

```python
from functools import lru_cache

@lru_cache(maxsize=128)
def compute_downstream_count(urn, lineage_graph):
    """Memoized downstream count computation."""
    count = 0
    for neighbor in lineage_graph.get(urn, []):
        count += 1 + compute_downstream_count(neighbor, lineage_graph)
    return count
```
